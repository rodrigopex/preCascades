DESAFIO 01

Você deve implementar a classe CPPList que herda da classe abstrata List todas as suas caracterícias. Lembre-se de reimplementar todos os métodos virtuais puros! No próprio código da lista está documentado o que cada método deve fazer. 

Dúvidas teórias e dicas de implementação podem ser encontradas no livro: http://people.cs.vt.edu/~shaffer/Book/C++3elatest.pdf
